expense tracker

i made a expense tracker (i call it expense base). it's built with tkinter and uses sqlite for database. well, just like everyone else i too don't like the widgets tkinter gives by default. recently, i did learn that there's something called custom tkinter that gives more modern looking widgets.

i wanted a solution asap so i used tkinter. i have also released a .exe for anyone on windows to download. the code and release is on github.
